HTML & CSS Cheatsheet

HTML

Tag Name Description
!DOCTYPE html Declaration Declares the file as HTML. All HTML files need to start with this declaration.
html Opening tag After the declaration, the file continues with this opening tag to write in HTML.
body Body This opens the body of the HTML document.
h1 Heading Element There are several heading elements that can be used, which include: h1, h2, h3, h4, h5, and h6. H1 is the most important heading, with h6 being the least important.
p Paragraph This tag allows you to construct the paragraph text.
a href HREF Attribute This tage using creates a hyperlink, which you can use to link a URL.
img Insert Image This tag allows you to insert an image. You can specify height and width dimensions. There is also an alt attribute that can be added to support accessibility for the reader.

CSS Selectors

Selector Name Description
p HTML Element Using p as an example, the selector refers to an HTML element and styles can be added to that element.
# id Selector Use a hash character (#) to refer to an id attribute classified in an element. The id needs to be unique, and can allow for more specific referencing.
. Class Selector Using a period (.) to select a class attribute.
* Universal Selector Use a star character(*) to select all HTML elements.

CSS Properties

Property Description
color: This property can apply colors in a wide range. Colors can be specified using a name, such as blue or Tomato. Colors can also be specified using RGB, HEX, HSL, RGBA, and HSLA values.
background-color: Apply a color to a specific background. Can also apply an image using the property 'background-image'.
border-style: There are many property values allowed for this property, such as dotted, dashed, solid and none.
margin: 4 ways to specify margin: margin-top, margin-right, margin-bottom, and margin-left.
padding: Similar to margin, this specifies the space around an element. There are 4 properties to specify padding: padding-top, padding-right, padding-bottom and padding-left.
text-transform: Can convert the text to uppercase and lowercase.
font-family: You can import fonts, use web-safe fonts as backups and distinguish between serif and sans-serif fonts.
